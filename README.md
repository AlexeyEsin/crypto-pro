## CryptoPro

Единое, асинхронное API для взаимодействия с КриптоПРО ЭЦП Browser Plug-In (IE8+)

``` js
CryptoService.call('getCertsList').then(function (certList) {
    console.log(certList);
});
```

Полифиллы необходимо подключать самостоятельно.
